Loja do Galo

Descrição do projeto

O projeto "Loja do Galo" é uma simulação de uma página de e-commerce desenvolvida com HTML e CSS puro, como parte de uma atividade prática da disciplina de Programação Web (XDES03) da UNIFEI.

A página apresenta um layout responsivo, com cabeçalho contendo barra de busca com filtro de categoria, cards de produtos organizados visualmente, cada um com imagem, nome, preço, descrição e botão de compra.

As fontes foram importadas do Google Fonts e aplicadas aos títulos e conteúdos para melhor apresentação visual. Todo o estilo foi implementado usando Flexbox, sem nenhuma biblioteca externa.

Este projeto demonstra domínio de estruturação HTML semântica, uso eficiente de CSS e princípios básicos de responsividade. O rodapé inclui um link para o GitHub do autor, simulando um ambiente profissional.
